# Automated-Indoor-Plant-Irrigation-System
This project involves designing an intelligent system for automatic irrigation of indoor plants. The system continuously monitors the soil moisture level, activates a water pump when the moisture level drops below a set threshold, and ensures that the plant receives adequate water. The system also provides real-time environmental data such as temperature, humidity, and sunlight exposure via sensors. Additionally, it alerts users with a sound signal when the water reservoir is low.

Components Used:
- Arduino UNO microcontroller,
- Soil Moisture Sensor for measuring soil hydration,
- Photoresistor for sunlight measurement,
- LM35DZ Temperature Sensor for ambient temperature monitoring,
- DHT11 Humidity Sensor for room humidity measurement,
- Water Level Sensor to detect the water level in the reservoir,
- Mini Water Pump (4-6V) for watering the plant,
- BD135 Transistor for controlling the water pump,
- 2x16 LCD Display with I2C for displaying sensor data,
- Buzzer for low water level alerts,
- 2xAA Battery Holder for powering the water pump,
- Wires, resistors, and capacitors for additional components.

System Functionality:
- Continuously monitors the soil moisture and activates the pump when required,
- Alerts the user when the water level in the reservoir is low using a buzzer,
- Displays real-time sensor readings (soil moisture, temperature, sunlight) on the LCD,
- Simple and efficient operation with minimal user interaction required.
